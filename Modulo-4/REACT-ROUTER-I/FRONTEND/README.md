# React + Vite


## React Router Dom

<BrowserRouter> = El que inicia todo el enrutado de la app.

<Routes> = El contenedor de las rutas.

<Route path="/" element={<Vista/>}> = La definición de que vista se va a renderizar en que ruta.

<Link to="/"> = Exactamente igual a la etiqueta <a href="/"> PERO no actualiza la web. (como en los forms)
